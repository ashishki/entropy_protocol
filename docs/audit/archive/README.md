# Audit Archive

This directory stores historical audit material that is no longer part of the
default context load.

| Directory | Contents |
|---|---|
| `phase_reviews/` | Historical Phase 1-8 review documents |
| `phase0/` | Phase 0 strategic, gate, and final sync documents |
| `p0_5/` | Phase 0.5 evidence-hardening plans and statistical reviews |
| `p0_6/` | Source manifest, Binance/P4 canary, and universe bootstrap records |
| `p0_7/` | Archive-only continuation and gate packet records |
| `p4/` | P4 coverage, conversion, and universe scaling records |
| `simbroker/` | SimBroker calibration planning and verification records |
| `data_stability/` | Data stability bootstrap, append, simulation, and archive records |
| `dispositions/` | D010 and T21-T24 closure/disposition packets |
| `legacy/` | Superseded audit reports and the pre-cleanup audit index |
| `legacy_prompts/` | Superseded audit prompt trio from the earlier review pipeline |

Archived files are retained for audit traceability. They are not active task
instructions unless a current document explicitly references them.
