# Paid Pilot Offer

Date: 2026-05-19
Status: internal_offer_hypothesis
Gate: `approve_internal_only`

## Offer Name

Bounded Public Source Utility Audit.

## Positioning

This is a paid pilot hypothesis for buyer discovery. It is not a public product
launch, not an externally approved report, not trading advice, and not a live signal service.

## Scope

Default pilot scope:

- one buyer;
- one fixed historical window;
- up to 3 public/operator-authorized sources;
- text-first analysis;
- media evidence only when already source-linked and human/operator accepted;
- deterministic market validation only for approved provider/proxy paths;
- one internal review call after delivery.

Out-of-scope by default:

- private or login-walled source capture;
- live monitoring;
- broker integration;
- channel ordering for public marketing;
- unreviewed transcript/OCR/chart claims;
- unsupported provider/proxy assumptions;
- investment recommendations or performance promises.

## Deliverables

| Deliverable | Description |
|---|---|
| Source utility report | Internal report covering coverage, clarity, extraction quality, outcome quality, risk quality, and limitations. |
| Metrics artifact | JSON or CSV-shaped summary of reviewed claims, outcomes, exclusions, and media blockers. |
| Evidence appendix | Source, provider, snapshot, and review refs for supported metric rows. |
| Robustness appendix | Horizon, provider, benchmark, sample-size, and review-coverage warnings. |
| Exclusion register | Unsupported assets, provider gaps, ambiguous claims, and unreviewed media rows. |
| Review call | 45-minute walkthrough focused on whether the report supports the buyer's internal decision. |

## Price Hypothesis

Discovery price hypothesis: USD 2,500-7,500 for the bounded pilot above.

This is a hypothesis to validate willingness to pay. It is not a published
price list. Larger source lists, custom media review, extra provider work,
or recurring runs require a separate scope decision.

## Turnaround

Target turnaround: 5-10 business days after scope lock, source authorization,
and provider/proxy assumptions are recorded.

Turnaround can extend when:

- source capture is incomplete;
- operator decisions are missing;
- provider/proxy mapping is unresolved;
- media requires human/operator review;
- requested outputs exceed the default scope.

## Buyer Commitments

The buyer must provide:

- public/operator-authorized source list;
- historical window;
- target decision the report should support;
- acceptable exclusions;
- point of contact for scope and review decisions;
- confirmation that the pilot is for internal research use.

## Success Gate

A pilot is successful only if at least three are true:

- buyer pays or explicitly approves a paid procurement path;
- buyer provides real sources and a real decision context;
- buyer uses the output in an internal research/compliance/product workflow;
- buyer asks for a repeat run, expanded scope, or integration discussion;
- buyer identifies missing output that would make the report materially more
  useful.

## Kill Criteria

Stop or pivot the offer if:

- buyers only want live alerts or investment instructions;
- buyers do not care about evidence provenance;
- buyers refuse to define a decision context;
- buyers need private-source access the product cannot legally support;
- buyers will not pay for bounded audit work after seeing the internal demo.

## External Boundary

The pilot deliverable remains internal-only unless a later external-ready gate
explicitly approves customer-facing delivery. Current gate decision is
`approve_internal_only`.
