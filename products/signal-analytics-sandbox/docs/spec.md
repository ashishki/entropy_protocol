# Signal Analytics Sandbox Spec

## MVP Inputs

- Public source identifier.
- Public message history accessible without bypassing controls.
- Signal extraction rule.
- Market data source for outcome calculation.

## MVP Outputs

- Signal ledger.
- Per-signal outcome table.
- Win/loss summary with assumptions visible.
- Drawdown/equity-style chart only when assumptions are explicit.
- Missing-data and ambiguity notes.

## Explicit Cuts

- No private Telegram group scraping.
- No marketplace.
- No influencer verification badge.
- No paid X API dependency before customer validation.
- No integration into Entropy Core phase-gate evidence.
